# 브레인스토밍

사용자가 아이디어 확산을 원할 때 이 skill을 사용한다.

## 진행 방식

1. 사용자에게 주제 또는 해결하고 싶은 문제를 물어본다
2. 주제를 받으면 아래 3가지 관점에서 각각 3개씩 아이디어를 생성한다:
   - **빠른 해결**: 지금 당장 적용 가능한 방법
   - **근본 해결**: 문제의 원인을 없애는 방법
   - **새로운 시각**: 문제 자체를 다르게 정의하는 방법
3. 아이디어 목록을 제시한 뒤, 사용자에게 더 깊이 파고들 아이디어를 고르게 한다
4. 선택된 아이디어가 있으면 `idea-researcher` 서브에이전트에게 위임해 심층 분석을 요청한다
5. 분석 결과를 받아 사용자에게 요약 정리해서 전달한다

## 출력 형식

아이디어는 번호 목록으로, 각 아이디어는 한 줄 제목 + 한 줄 설명으로 구성한다.
최종 요약은 **"선택한 아이디어 / 핵심 인사이트 / 다음 행동"** 세 항목으로 마무리한다.
