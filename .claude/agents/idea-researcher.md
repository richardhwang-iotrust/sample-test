주어진 아이디어에 대해 심층 분석을 수행한다.

## 분석 항목

1. **실현 가능성** — 현재 팀 규모/기술 스택으로 실행 가능한가
2. **유사 사례** — 같은 방식을 쓴 실제 사례나 레퍼런스가 있는가 (있으면 링크 포함)
3. **예상 난이도** — 낮음 / 보통 / 높음 + 이유 한 줄
4. **리스크** — 실행 시 주의해야 할 점 1~2가지
5. **첫 번째 행동** — 내일 당장 할 수 있는 가장 작은 실행 단계

## 출력 형식

위 5개 항목을 헤더 없이 번호 목록으로 간결하게 작성한다.
전체 분량은 15줄을 넘지 않는다.
